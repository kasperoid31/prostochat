## 📝 Живая книга

**Эта книга никогда не будет дописана до конца.**

«просточат» — это живая история. Каждый день в ней могут появляться новые страницы, новые детали, новые имена.

Кто-то вдруг вспомнит забытую шутку. Кто-то найдёт старый скриншот. Кто-то захочет рассказать свою версию событий.

**И это нормально.**
